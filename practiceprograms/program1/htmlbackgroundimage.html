background element for div element:

you can specifiy background images
for any visibal html element.
in this example background image
is specified for a p element.
by default, the background-image
will repeat itself in the direction(s)
where is it smaller than the element
where is it specified.(try resizing the
browser window to see how the
background image behaves.
